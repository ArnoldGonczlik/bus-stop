# Bus Stop

A minimal first-person browser game built with Three.js and Vite.

## Description

Wait at a bus stop, watch the bus arrive, and board it to win. Simple FPS controls with mouse look and WASD movement.

## Tech Stack

- **Vite** - Build tool and dev server
- **Three.js** - 3D rendering
- **Plain JavaScript** - No frameworks

## Getting Started

```bash
npm install
npm run dev
```

Open your browser to the URL shown (typically http://localhost:5173).

## Controls

- **Click** - Start game and lock pointer
- **Mouse** - Look around
- **W/A/S/D** - Move
- **ESC** - Unlock pointer

## Gameplay

1. Click to start
2. Look around and explore
3. Wait for the bus to arrive
4. Walk forward into the bus door
5. Win!

## Build

```bash
npm run build
```

Preview the build:

```bash
npm run preview
```
